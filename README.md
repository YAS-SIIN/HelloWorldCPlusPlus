# HelloWorldCPlusPlus

This is a small C++ practice project I created while learning the C++ programming language.

The project is a simple console application and includes examples of:

- printing output to the console
- working with variables and conditions
- using `for`, `while`, and `do while` loops
- pointers and references
- basic class usage across multiple source files

It was built as a learning exercise to understand core C++ syntax and programming concepts.
